# Supported Skill Groups

## HR-scopes

Employee lifecycle workflows (onboarding, offboarding, leave, role changes, performance reviews). Prefer a skill when the request matches a known HR workflow; otherwise use a custom script for ad-hoc analysis or one-off tool tasks.

### Tools
- BambooHR
- Gmail
- Google Calendar
- Jira
- Lattice
- Slack
- Candidate Tracker

### Skills
- Daily New Hires Digest
- Leave & Absence Management
- Offboard Employee
- Offboarding Queue Review
- Onboard New Hires
- Performance Review Cycle
- Probation Check-in Reminders
- Role Change + Access Review

## Recruitment-scopes

Candidate pipeline coordination workflows (scheduling, reminders, stage updates). Use this group when the request is primarily about candidates, interviews, and recruiter coordination.

### Tools
- Candidate Tracker
- Google Calendar
- Jira
- Slack

### Skills
- Chase Interview Feedback
- Schedule Candidate Interviews
- Candidate Pipeline Review

## Procurement-scopes

Purchase requests and vendor onboarding workflows. Use this group when the request is primarily about procurement coordination and purchase/vendor tracking.

### Tools
- Google Calendar
- Jira
- Slack

### Skills
- Create Purchase Request
- Vendor Onboarding Request
